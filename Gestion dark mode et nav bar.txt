Gestion du dark mode
Pour utiliser le dark mode sur la page il faut :
1) inclure le fichier CSS light_style sur sa page HTML en premier fichier CSS sur la page : link id="theme" rel="stylesheet" href="/asset/styles/light_style.css"/
2) inclure le fichier JS après le </body> mais avant </html> afin de pouvoir utiliser la fonction qui permet de changer de fichier 
CSS :  <script src="asset/js/index.js"></script>
3) donner la classe darkMode et lightMode en fonction de ce qu'on veut faire apparaitre. Attention de bien respecter la casse
et l'orthographe.
